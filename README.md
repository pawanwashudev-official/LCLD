## ✨ Key Features & Major Enhancements

### 🆕 Elegant & Simplified UI
We've completely redesigned the interface to be beautiful, intuitive, and modern. Setup is now a breeze—go from installation to full protection in under 60 seconds.

### 🚨 Unstoppable Siren (Enhanced Ring)
The `ring` command is now a true siren. 
- **Auto-Max Volume:** Instantly forces device volume to 100%.
- **Anti-Mute:** If a thief tries to lower the volume, LCLD automatically pulls it back to maximum.
- **Unlock to Stop:** The siren **will not stop** until the device is successfully unlocked by the owner.

### ⚡ Integrated Server Support
Unlike the original FMD, **LCLD does not require an additional app** (like Push Distributor) to receive server commands. We've built the communication layer directly into the app for a lighter, more reliable experience.

### 🛑 THEFT MODE (The Ultimate Countermeasure)
The new `theft` command is a proactive security protocol that triggers multiple actions simultaneously:
- **Instant Lockdown:** Locks the device immediately.
- **Continuous Tracking:** Starts aggressive location reporting.
- **Siren Activation:** Triggers the unstoppable siren.
- **Persistence:** Even if the thief restarts the phone, the siren and lock will resume **instantly** upon boot. There is no escape.

### 📸 Remote Camera & Precision Locating
- **Remote Snap:** Capture high-quality photos from front/back cameras.
- **GPS + Cell Tower:** Locate your device even when GPS is disabled or signal is weak.

## 🛠️ Getting Started

1. **Download:** Get the latest APK from [Releases](https://github.com/pawanwashudev-official/LCLD/releases).
2. **Setup PIN:** Set your secure 4-6 digit PIN for SMS commands.
3. **Grant Permissions:** Enable Location, Camera, and Device Admin for full protection.
4. **Trusted Contacts:** (Optional) Add numbers that can control the device without a PIN.

## 🔒 Security & Privacy
LCLD is built with a **Security-First** mindset. 
- **No Tracking:** We don't track your location. Your data is only shared with the channels *you* configure.
- **Encrypted Communication:** Server communication is secured via industry-standard encryption.
