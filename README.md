# LCLD - Locate & Control Lost Device

Developed by **Pawan Washudev** ([@pawanwashudev-official](https://github.com/pawanwashudev-official))

LCLD is a rebranded version of the open-source FMD project, providing a secure and privacy-focused way to locate and control your devices.

## 🤖 Automatic Builds
Download the latest APKs from the **Actions** tab of this repository.
