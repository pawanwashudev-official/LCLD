# LCLD - Locate & Control Lost Device

**LCLD** is a secure, open-source Android application for remotely finding and managing your device.

Developed by **Pawan Washudev** ([@pawanwashudev-official](https://github.com/pawanwashudev-official))

---

## ❤️ Huge Thanks to Original Developer
This project is a rebranded version of the amazing **Find My Device (FMD)** app by **Nulide**. We extend our deepest gratitude to the FMD contributors for their high-quality open-source foundation.

*   Original Project: [gitlab.com/fmd-foss/fmd-android](https://gitlab.com/fmd-foss/fmd-android)
*   Original Developer: [Nulide](https://fmd-foss.org)

---

## 🤖 Automatic Builds
You can manually trigger and download APK builds from the **Actions** tab of this repository.

## 🛠️ Get Started
1. **Clone:** git clone https://github.com/pawanwashudev-official/LCLD.git
2. **Build:** Open in Android Studio and build the prod flavor.
3. **Setup:** Set your allowed contacts or connect to an FMD Server.
